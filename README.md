# FilesDemo
📌 Описание
Данная практическая работа предназначена для ознакомления с механизмами импорта данных из Excel в WPF-приложение, их отображения и редактирования в DataGrid, а также экспорта полученных данных в формат PDF.

В ходе работы используются библиотеки EPPlus (работа с Excel) и iText7 (генерация PDF). Приложение позволяет:

Загрузить данные из .xlsx файла в DataGrid

Добавлять новые строки

Удалять выбранные строки

Редактировать ячейки непосредственно в таблице

Сохранить итоговую таблицу в PDF-документ

🛠️ Требования к окружению
Visual Studio 2017/2019/2022 (с поддержкой .NET Framework 4.7.2)

.NET Framework 4.7.2 SDK

Установленные пакеты NuGet (указываются ниже)
